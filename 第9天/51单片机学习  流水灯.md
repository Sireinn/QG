
# 原理
- 点亮D1然后延时依次点亮
- 管脚由低到高---循环操作+移位操作
# 实现手段
## 宏定义
	**# define  LED_PORT  P2 //使用宏定义将p2端口定义
宏定义是文本的替换，不要加；不要加等号
	
## for循环+位移
- 全都取然后动零
**u8 i=0；**
**while（1）**
**{**
	**for（i=0；i<）8;i++)**
	**{**
		**LED_PORT=0xfe<<i;**  
	**}**
**}**
----->移动零位，一位，，，，除了第一个p20低电平其他高电平然后除了p21低电平其余高电平，
![[Pasted image 20260210121449.png]]

再补零 让第一指针也为零----->不行
![[Pasted image 20260210121741.png]]
全部取零然后让1位移，位移相当于那一串数位移，然后第一位空了之后自动补零达成1个相反的结构，我们再利用c语言中取反即可

---- 
- 全都取0然后动1再取反
	**u8 i=0；**
	**while（1）**
	**{**
		**for（i=0；i<）8;i++)**
		**{**
			**LED_PORT=~（0x01<<i）;**   //i=0；D1亮  ，i=1，
			**delay_10us(50000);**
		**}**
	**}**

## 利用库文件自带的左移和右移动
**intrins.h库**
![[Pasted image 20260210122847.png]]

	#include "intrins.h"

	_cror----->右移
	-crol----->左移

### 不同之处
- 相当于排队，将最前面挤到最后一位
- 而for循环加上<<的左移是将整个数组往前挪，空出来的一位补上零

	## 实现
		#include"intrins.h"
	# define  LED_PORT  P2 //使用宏定义将p2端口定义
		
		void main()
	{
		u8 i=0；
		LED_PORT=~0x01;
		while（1）
		{
			for（i=0；i<7）;i++)
			{
				LED_PORT=_ crol_(LED_PORT,1);//(要移动的数,移动的距离)
				delay_10us(50000);
			}
			for（i=0；i<7）;i++)
			{
				LED_PORT=_ cror_(LED_PORT,1);//(要移动的数,移动的距离)
				delay_10us(50000);
			}
		}
		
	}
